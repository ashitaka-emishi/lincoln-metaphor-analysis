# Metaphor Reference Cards

*Quick reference for annotation and analysis. One card per cluster.*

## Cluster 01 — Nation as Organism / Body

| Field | Value |
|-------|-------|
| **Cluster ID** | `cluster_01_body_organism` |
| **Source domain** | wound, healing, birth, severance, disease |
| **Target domain** | the American Union |
| **Primary fantasy type** | `wound_and_healing` |
| **Primary violence logic** | `restorative` |
| **Obligatory frame** | Yes — wound demands physician |
| **Exit condition** | Yes — wounds heal |
| **What it hides** | Enslaved people absent as healers; wound's human authors abstracted; healing conceals depth of damage |
| **Key instance** | "binding up the nation's wounds" — Second Inaugural, 1865 |
| **Lincoln-specific note** | NO disease_and_purification. The wound can heal because no pathogen is present. This is the exit condition Hitler lacked. |
| **Hitler parallel** | Racial body (Volksgemeinschaft) — but Hitler adds disease logic Lincoln does not |

## Cluster 02 — Union as Covenant / Oath

| Field | Value |
|-------|-------|
| **Cluster ID** | `cluster_02_covenant_oath` |
| **Source domain** | sworn oath, contract, sacred bond |
| **Target domain** | the constitutional compact |
| **Primary fantasy type** | `oath_and_obligation` |
| **Primary violence logic** | `obligatory` |
| **Obligatory frame** | Yes — oath-sworn cannot withdraw; Lincoln bound by inaugural oath |
| **Exit condition** | Yes — covenant fulfilled when Union restored |
| **What it hides** | Covenant's parties implicitly white; enslaved people are object of covenant terms, not parties; Declaration vs. Constitution tension |
| **Key instance** | "I take the official oath to-day with no mental reservations" — First Inaugural, 1861 |
| **Key suppression event** | Greeley Letter (doc_012): covenant deployed against emancipation. Track political flexibility. |
| **Hitler parallel** | Blutgemeinschaft (blood bond) — but racial-biological, not contractual-rational |

|-------|-------|
| **Cluster ID** | `cluster_03_experiment_proposition` |
| **Source domain** | logical proof, scientific test |
| **Target domain** | democratic self-government |
| **Primary fantasy type** | `experiment_and_proof` |
| **Primary violence logic** | `evidentiary` |
| **Obligatory frame** | Yes — experiment underway must be completed |
| **Exit condition** | Yes — proposition proven (or disproven) |

## Cluster 04 — War as Birth / New Creation

| Field | Value |
|-------|-------|
| **Cluster ID** | `cluster_04_birth_creation` |
| **Source domain** | labor, nativity, generative act |
| **Target domain** | the refounding of the nation |
| **Primary fantasy type** | `birth_and_creation` |
| **Primary violence logic** | `generative` |
| **Obligatory frame** | Yes — birth cannot stop midway |
| **Exit condition** | Yes — child grows up; new nation born |
| **What it hides** | Black soldiers as literal embodiment of birth/sacrifice (erased); women absent from birth labor; new nation's inheritance of old nation's debts concealed |
| **Key instance** | "brought forth... new birth of freedom" — Gettysburg, 1863 |
| **Lincoln-specific note** | The "new birth of freedom" positions the post-war nation as *different* from pre-war, not merely restored. This is the cluster that makes emancipation possible as birth, not merely as healing. |
| **Hitler parallel** | Aryan rebirth / Third Reich as new creation — but racial-exclusive; Lincoln's birth is in principle inclusive |

## Cluster 05 — Founding Fathers as Inheritance

| Field | Value |
|-------|-------|
| **Cluster ID** | `cluster_05_fathers_inheritance` |
| **Source domain** | patrimony, lineage, ancestral debt |
| **Target domain** | obligation to the founders |
| **Primary fantasy type** | `ancestral_debt` |
| **Primary violence logic** | `obligatory` |
| **Obligatory frame** | Yes — you cannot choose your inheritance; ancestral debt is given, not negotiated |
| **Exit condition** | Yes — debt paid when founding project preserved/completed |
| **What it hides** | Founders' ambivalence on slavery; enslaved people cannot inherit the civic legacy; idealization conceals historical complexity |
| **Key instance** | "our fathers brought forth" — Gettysburg; "the same men who framed the Government" — Cooper Union |
| **Lincoln-specific note** | Cooper Union (doc_007) is the best single document: Lincoln names specific founders as legal authorities, treating their votes as binding precedent. |
| **Hitler parallel** | Germanic racial ancestors — but blood-inherited, not civic; Lincoln's lineage can be joined through oath |

| Field | Value |
|-------|-------|
| **Cluster ID** | `cluster_06_providence_theodicy` |
| **Source domain** | God's judgment, punishment, theodicy |
| **Target domain** | the war's cause and meaning |
| **Primary fantasy type** | `punishment_and_theodicy` |
| **Primary violence logic** | `punitive` + `obligatory` |
| **Obligatory frame** | Yes — at maximum; divine will cannot be resisted |
| **Exit condition** | Yes — punishment runs its course; God's purposes accomplished |
| **What it hides** | Human agency and political failure displaced onto divine will; specific agents of slavery lose particular moral responsibility in cosmic guilt distribution; lincoln_non_agent at maximum |
| **Key instance** | "The Almighty has His own purposes" — Second Inaugural, 1865 |

| **Hitler parallel** | Providence / historical destiny — but Hitler's Providence is legible and selecting; Lincoln's is inscrutable and punishing both sides |

## Valley Campaign Cross-Pollination Table

*The Shenandoah Valley / Valley Campaign scenarios (1862 and 1864) fall in Phase 3 (early) and Phase 4 (late) of Lincoln's diachronic arc. The proximate rhetorical context for all Valley scenarios is the period bracketed by the Greeley Letter (August 1862) and the Gettysburg Address (November 1863).*

| Valley scenario / period | Lincoln phase | Dominant Lincoln clusters | Guilt distribution | Key Lincoln document in period |
|--------------------------|--------------|--------------------------|-------------------|-------------------------------|
| Jackson's Valley Campaign (Spring 1862) | Phase 3 (Obligation) | cluster_02 (covenant), cluster_01 (body) | External | First Inaugural background; July 4 Message 1861 |
| Early's Raid (Summer 1864) | Phase 4-5 transition | cluster_06 (providence), cluster_04 (birth) | Distributed → Cosmic | Blind Memorandum, Hodges Letter |

*Note: The Valley campaigns produced no Lincoln speeches directly about the Valley. The relevant context is Lincoln's general war rhetoric in each period. For Spring 1862: the covenant cluster is dominant — Lincoln is obligated to preserve the Union. For Summer 1864: the providence/theodicy cluster is developing — Lincoln is an instrument of events he cannot control.*

*The Greeley Letter (August 1862) is the single most important rhetorical document bracketing the Valley Campaign of Spring 1862: it shows Lincoln's covenant logic at maximum, with Freedom subordinated to Union, written in the shadow of military reverses including Jackson's Valley victories.*
